GCC、GDB食用方法

发表于2022-03-06|更新于2023-09-08|实用技能
|字数总计:241

GCC

-o自定义目标文件

-Wall自动提示一些出错警告的信息

-L 链接一个库文件，-L.加上的点意义为将本目录文件作为查找库的根目录

GDB

在GCC编译时需要添加一个参数：

如无报错，即可启动GDB：

GDB指令

break 10在第十行设置断点。

break test在test函数处设置断点

s / step下一步

c / continue当程序在某一断点处停止后，用该指令可以继续执行，直至遇到断点或者程序结束

u / until 运行程序直到退出循环体

r / run运行

q / quit结束调试

p / print显示某个变量的值

p *array@len 输出数组

enable 恢复失效的断点

disable 使断点失效

clear 清除断点

bt / backtrace 查看堆栈信息

watch监测某变量或表达式，当发生变化时停止
